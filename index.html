Começar

O que é HTML, CSS e JS?

HTML

HTML é a linguagem de marcação que define a estrutura do seu conteúdo web. É a base de qualquer página web.

<!DOCTYPE html>
<html>
  <head>
    <title>Minha página</title>
  </head>
  <body>
    <h1>Olá, Mundo!</h1>
  </body>
</html>

Quando usar: Para criar a estrutura básica e organizar o conteúdo de páginas web.

CSS

CSS é a linguagem de estilização que controla a apresentação visual dos elementos HTML.

.button {
  background: var(--primary);
  color: white;
  padding: 1rem 2rem;
  border-radius: 4px;
}

Quando usar: Para estilizar elementos, criar layouts e tornar as páginas visualmente atraentes.

JavaScript

JavaScript é a linguagem de programação que adiciona interatividade e comportamento dinâmico às páginas.

const button = document.querySelector('.button');
button.addEventListener('click', () => {
  alert('Botão clicado!');
});

Quando usar: Para adicionar interatividade, manipular dados e criar funcionalidades dinâmicas.
